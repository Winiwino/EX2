# Козлова Анастасия

## Spring / вариант задания 2

## Описание проекта
Проект представляет собой REST API путеводителя по городу, реализованное на Spring Boot.

Сервис позволяет:
- получить список достопримечательностей в указанном радиусе от пользователя;
- фильтровать список по категории и средней оценке;
- сортировать результаты по расстоянию, рейтингу или названию;
- просматривать информацию по конкретной достопримечательности;
- просматривать отзывы по конкретной достопримечательности;
- добавлять оценку достопримечательности;
- добавлять отзыв о посещении достопримечательности.

В качестве базы данных используется H2. Для создания структуры БД и начального наполнения используются миграции Flyway.

## Подготовительные действия
Для успешной работы проекта необходимо установить:
- JDK 21;
- Maven или использовать Maven Wrapper, который уже находится в проекте;
- Postman или другой HTTP-клиент для проверки REST API.

## Информация о доступах
Параметры встроенной базы H2:
- URL: `jdbc:h2:mem:guide`
- Username: `sa`
- Password: пустой

Если включена H2 Console:
- URL консоли: [http://localhost:8080/h2-console](http://localhost:8080/h2-console)

## Как запустить проект
1. Открыть терминал в корне проекта.
2. Убедиться, что установлен JDK 21.
3. Запустить приложение командой:

```powershell
.\mvnw.cmd spring-boot:run
```

После запуска приложение будет доступно по адресу:
- [http://localhost:8080](http://localhost:8080)

Для запуска тестов использовать команду:

```powershell
.\mvnw.cmd test
```
